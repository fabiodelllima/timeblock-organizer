# Formação de Hábitos

## Fundamento Teórico

Ver [Filosofia: Hábitos Atômicos](../../04-specifications/philosophy/atomic-habits.md) para aplicação completa no TimeBlock.

## Pesquisa Científica

### Tempo para Automatização

**Lally et al. (2010)** - European Journal of Social Psychology

**Resultado:** Média 66 dias para hábito se tornar automático (variação: 18-254 dias)

**Implicação:** Horizonte 8 semanas do TimeBlock alinha com ciência.

### Loop do Hábito

**Duhigg (2012)** - "The Power of Habit"

```terminal
Cue -> Routine -> Reward -> Craving
```

**Implementação TimeBlock:**

- Cue: Notificação horário agendado
- Routine: Executar hábito
- Reward: Marcar completo + estatísticas
- Craving: Ver progresso/streaks

### Implementation Intentions

#### **Gollwitzer (1999)**

Especificar "quando/onde" aumenta 2-3x taxa execução.

**Formato:** "Eu vou [COMPORTAMENTO] às [HORÁRIO] em [LOCAL]"

**TimeBlock:** Campo `scheduled_start_time` implementa isso.
