# Primeiros Passos

## Instalação
```shell
pip install timeblock-organizer
```

## Primeiro Hábito
```shell
timeblock habit add "Meditar" --recurrence "FREQ=DAILY" --duration 15
```

## Ver Agenda
```shell
timeblock habit list
```
