# Tutorial: Primeiro Hábito

## Objetivo

Criar seu primeiro hábito e executá-lo com timer.

## Passo 1: Instalar

```shell
pip install timeblock-organizer
```

## Passo 2: Criar Hábito

```shell
timeblock habit add "Meditar" --recurrence "FREQ=DAILY" --duration 15
```

## Passo 3: Ver Hoje

```shell
timeblock habit list --today
```

## Passo 4: Iniciar Timer

```shell
timeblock timer start <id>
```

## Passo 5: Finalizar

```shell
timeblock timer stop
```

Pronto! Primeiro hábito concluído.
