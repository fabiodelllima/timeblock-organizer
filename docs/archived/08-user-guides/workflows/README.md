# Workflows por Versão

Workflows detalhados mostrando inputs e outputs reais para cada versão do TimeBlock Organizer.

## Workflows Disponíveis

### Atual (Funcional)

**[v1.0 Baseline](v1.0-baseline.md)**

- Setup e inicialização
- Adicionar eventos simples
- Listar com filtros
- Validações e erros
- Avisos de conflito (não-bloqueantes)

### Futuro (Planejado)

**v2.0 Development** (70% funcional)

- Criar rotinas com hábitos
- Adicionar tarefas únicas
- Timer para rastreamento em tempo real
- Tags e categorização
- Views de agenda (hoje, semana)

#### **Features Planejadas**

- **v1.1:** Event Reordering com propostas inteligentes
- **v1.2:** Relatórios e analytics
- **v2.0:** TUI (Textual User Interface)

## Como Usar Estes Workflows

Cada workflow mostra:

- **Input Usuário:** Comandos exatos digitados
- **Output Sistema:** O que você verá no terminal
- **Mudanças Estado:** O que acontece no banco
- **Notas:** Detalhes importantes ou pegadinhas

### Notação

```terminal
$ comando                    # Input usuário (o que você digita)
Texto output                 # Output sistema (o que você vê)
# Comentários                # Notas explicativas
```

### Suporte a Formatos

TimeBlock suporta formatos brasileiros de tempo:

- `9h` ou `09:00` -> 9:00 AM

- `14h30` ou `14:30` -> 2:30 PM

- `23h45` ou `23:45` -> 11:45 PM
